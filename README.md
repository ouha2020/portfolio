# Cloud Native DevOps Portfolio

React + Vite + TypeScript single-page portfolio site for presenting cloud-native DevOps work.

## Highlights

- Cloud-native DevOps portfolio layout
- Chinese/Japanese bilingual content model
- CI/CD, Kubernetes, observability, service mesh, and certification sections
- Interactive code browser with selected project directories and implementation notes
- Local official logo assets for key technologies

## Run Locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
